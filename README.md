# 🤰 Gestanea — Pregnancy & Baby Care App

A modern mobile application designed to support pregnancy tracking, baby care, health monitoring, appointments, and maternal well-being in one unified platform.

Built with a **clean, scalable architecture** and a **modern neumorphic UI** experience.

---

## 🌐 Overview

Gestanea is a full-featured maternal health platform that helps users:

- Track pregnancy progress week by week
- Monitor baby development
- Log health data (symptoms, vitals, mood)
- Book and manage appointments
- Access educational content
- Shop relevant maternal products
- Communicate within a supportive ecosystem

The goal is to make pregnancy **simpler, safer, and more informed**.

---

## ⚙️ Tech Stack

### 📱 Mobile App
- Flutter (clean modular architecture)
- Bloc (state management)
- Neumorphic UI design system

### ☁️ Backend & Services
- Supabase (Authentication, Database, OTP system)
- FastAPI (custom backend services)
- Firebase Messaging (push notifications)
- Email services (verification + alerts)

### 🗺️ External Integrations
- OpenStreetMap (geolocation + nearby services)

## 🧱 Architecture

The app follows a **modular clean architecture approach**:

- Feature-based structure
- Separation of UI, logic, and data layers
- Scalable and maintainable design

Each module is independent:
- Auth
- Pregnancy tracking
- Baby care
- Health logs
- Marketplace
- Profile system
